# 开发支持模块

该模块用于支持简单的网站开发，集成了常用的功能

## 功能
- 自动API
- 网站设置
- 网站模板
- 用户系统
- 多角色权限系统
- 文件上传辅助
- 验证码生成辅助
- 条形码生成辅助
